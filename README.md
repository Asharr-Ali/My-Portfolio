## 🧑‍🔧 Getting Started

Follow these steps to run the project locally:

### 📦 Clone the repository

```bash
git clone https://github.com/Asharr-Ali/portfolio.git
cd portfolio
npm install
npx tailwindcss -i ./src/input.css -o ./dist/output.css --watch
start index.html

/portfolio/
├── index.html
├── /assets/
├── /css/
└── tailwind.config.js
